DHCP Finder
=========
A tool that can help you check whether there are other unauthorized DHCP routers on your local network  
This program takes more than ten seconds to start, which is normal. It is elevating privileges, otherwise scanning would not be possible  
This program only supports wired networks; scanning for wireless networks is meaningless  
  
This program cannot detect any DHCP servers in certain network environments:  
>1.Your computer is set to a static IP
>>2.Your computer's static IP is not within your router's DHCP address pool
